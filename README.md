CDマネージャー
=============

CDトレイを開いたり閉じたりするコマンドラインプログラム。  
ランチャープログラムと連携すると便利。  
個人的にCLaunchおすすめ。

- [Pyonkichi's page](http://hp.vector.co.jp/authors/VA018351/)

インストール方法
------------------

適当なところに解凍してお使いください。

アンインストール方法
------------------

レジストリ等使っておりませんので、そのままゴミ箱に入れて削除してください。

使い方
--------

### ドライブの一覧を表示

```
C:\>CDManager list
D
N
```

### トレイを開く

```
C:\>CDManager close
```

### トレイを閉じる

```
C:\>CDManager close
```
